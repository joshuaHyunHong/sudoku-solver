Simply looks at each unsolved cell and determines all possible answers for each cell. For instance, if a cell has 123 then the cell answer could be a 1 or a 2 or a 3. Achieves this by looking at cell values in the same row, column, and box.

Let's say that there's a box with cell 123 and another cell of just 3. The 3 ought to not exist since there can't possibly be another 3 in the same box. What has happened here is that the code indirectly solved for that cell and has not yet registered that 3 in the box, so simply press the button again and the 123 should become a 12. It may help to press the button multiple times every time you solve a cell, as the code may indirectly solve some additional cells for you.

Please note that this solver isn't meant to automatically solve sudokus since I made this with the intent of solving HARD sudokus that force you to utilize varrying advanced techniques and some guess-work.
